01. Загальні положення

Ця Політика конфіденційності описує, як Infinite Union («ми», «нас», «наш») збирає, використовує та захищає персональні дані користувачів веб-сайту ceoinfiniteunion-cell.github.io/iuwebsite та пов'язаних сервісів, включаючи Telegram-бот @infiniteunion_bot.

Використовуючи наш сайт або бот, ви погоджуєтесь з умовами цієї Політики. Якщо ви не погоджуєтесь — будь ласка, припиніть використання наших сервісів.

Ми поважаємо вашу приватність і зобов'язуємось обробляти персональні дані відповідально, законно та прозоро відповідно до Закону України «Про захист персональних даних» та Регламенту ЄС 2016/679 (GDPR).

02. Які дані ми збираємо

Ми збираємо лише ті дані, які необхідні для надання наших послуг:

Дані, які ви надаєте добровільно:

  • Ім'я або назва компанії
  • Номер телефону
  • Telegram username або інші контактні дані
  • Тип проєкту та бюджет
  • Дедлайн та опис проєкту
  • Будь-яка інша інформація, яку ви надаєте у формі зворотного зв'язку

Дані, що збираються автоматично:

  • IP-адреса
  • Тип браузера та операційна система
  • Час та дата відвідування
  • Сторінки, які ви переглядали
  • Джерело переходу на сайт

Дані з Telegram:

  • Telegram ID користувача
  • Username у Telegram
  • Повідомлення, надіслані боту в рамках квізу

03. Як ми використовуємо дані

Зібрані дані використовуються виключно для таких цілей:

  • Обробка заявок — для зв'язку з вами та обговорення вашого проєкту
  • Надання послуг — для виконання замовлень та договорів
  • Комунікація — відповіді на запити, надсилання комерційних пропозицій
  • Покращення сервісу — аналіз використання для поліпшення якості
  • Дотримання зобов'язань — виконання вимог законодавства

Ми ніколи не продаємо, не передаємо та не здаємо в оренду ваші персональні дані третім особам з маркетинговою метою.

04. Зберігання та захист даних

Ваші дані зберігаються на захищених серверах Railway.app (США, сертифікований хмарний провайдер) з використанням шифрування даних у стані спокою та при передачі.

Заходи безпеки:

  • Шифрування HTTPS/TLS для всіх з'єднань
  • Зашифрована база даних PostgreSQL
  • Обмежений доступ — лише авторизований персонал
  • Регулярне резервне копіювання даних
  • Моніторинг безпеки 24/7

Терміни зберігання:

  • Заявки та контактні дані — 3 роки з моменту отримання або завершення співпраці
  • Дані аналітики — 26 місяців
  • Резервні копії — 90 днів

Після закінчення терміну зберігання дані видаляються або знеособлюються безповоротно.

05. Передача даних третім особам

Ми можемо передавати ваші дані обмеженому колу третіх осіб виключно в рамках надання послуг:

  • Railway.app — хостинг та інфраструктура (США)
  • Telegram — для роботи бота (США/ЄС)
  • GitHub Pages — хостинг сайту (США)
  • Google Analytics — аналітика відвідуваності (якщо підключено)

Усі треті сторони зобов'язані дотримуватись відповідних стандартів захисту даних. Ми не передаємо дані державним органам, крім випадків, передбачених законом.

06. Файли cookie

Наш сайт використовує мінімальну кількість файлів cookie для забезпечення базової функціональності:

  • Технічні cookie — необхідні для роботи сайту, не потребують згоди
  • Аналітичні cookie — для розуміння поведінки відвідувачів (анонімно)

Ви можете відключити cookie у налаштуваннях вашого браузера. Це може вплинути на функціональність деяких елементів сайту.

Ми не використовуємо рекламні або відстежувальні cookie третіх сторін.

07. Telegram-бот

Наш Telegram-бот @infiniteunion_bot збирає дані виключно в рамках квізу для оформлення заявки:

  • Telegram ID та username — для ідентифікації та зворотного зв'язку
  • Відповіді на запитання квізу — для формування заявки
  • Час взаємодії — для адміністративних цілей

Дані, надіслані через бота, зберігаються в захищеній базі даних і доступні лише адміністраторам компанії. Ви можете запросити видалення своїх даних, написавши нам.

Взаємодія з ботом регулюється також Політикою конфіденційності Telegram.

08. Ваші права

Відповідно до законодавства про захист персональних даних, ви маєте такі права:

  • Право на доступ — отримати копію своїх персональних даних
  • Право на виправлення — виправити неточні або застарілі дані
  • Право на видалення — вимагати видалення своїх даних («право бути забутим»)
  • Право на обмеження обробки — обмежити обробку ваших даних
  • Право на перенесення даних — отримати дані у машиночитаному форматі
  • Право на заперечення — заперечити проти обробки даних
  • Право на скаргу — подати скаргу до наглядового органу

Для реалізації будь-якого з цих прав напишіть нам на ceoinfiniteunion@gmail.com або в Telegram @infiniteunion_manager. Ми відповімо протягом 30 днів.
