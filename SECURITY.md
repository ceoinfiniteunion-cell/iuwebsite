# Security Policy

## Reporting a Vulnerability

Do not open a public GitHub issue for security vulnerabilities.

Contact privately:
- Telegram: [@infiniteunion_manager](https://t.me/infiniteunion_manager)
- Response time: within 24 hours

## Security measures

- Content Security Policy via Vercel headers (planned)
- No secrets in client-side code
- All form submissions validated server-side
- Honeypot field on contact form
- Rate limiting on AI constructor endpoint
